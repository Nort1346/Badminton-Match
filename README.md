# Badminton Match
A comprehensive app for tracking and managing badminton matches.

# Features
- Manage Points and Sets
- Set Player Names
- Reset Scores
- Switch Sides
- Point Summary (with Speech Synthesis)
- Custom Rules
- Match History
- Serving Indicator
- Multi-language Support (English, Polish, and German)

# Install And Run

### Install badminton-match package
```bash

```
### Run the app
```bash
npm install
npm preview
```
